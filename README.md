# Customer_Behavior_Analysis
This project analyzes customer purchasing behavior to identify trends, discount impact, and product performance. The goal is to derive business insights that can help improve marketing strategies and sales decisions.

🗄 PostgreSQL

📊 Data Analytics Project
📌 Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The objective is to extract meaningful insights from raw data by performing data cleaning, exploratory data analysis (EDA), database querying, and dashboard visualization.

The project follows industry-standard practices and showcases skills required for a Data Analyst role.

.>Raw dataset loaded using Python

🔄 Project Steps
1️⃣ Data Loading

Imported dataset into Python

Inspected data structure

Checked data types and basic statistics

2️⃣ Exploratory Data Analysis (EDA)

Analyzed distributions and trends

Identified missing values and outliers

Generated summary statistics

Created visualizations for better understanding

3️⃣ Data Cleaning

Handled missing values

Removed duplicates

Converted incorrect data types

Performed feature engineering where necessary

4️⃣ SQL Analysis

Uploaded cleaned data to database (PostgreSQL)

Wrote SQL queries for:

Aggregations

Filtering

Grouping

Joins

Extracted business insights using SQL

5️⃣ Power BI Dashboard

Built interactive dashboard

Added slicers for filtering

Created KPI cards, bar charts, and trend visuals

Designed clean and user-friendly layout
